# Inventory Management System using SQL

## Tools Used
- Oracle SQL

## Project Description
Developed an inventory management system using Oracle SQL to manage product stock, sales, and inventory updates efficiently.

## Features
- Product stock management
- Inventory tracking
- Low stock identification
- Highest selling product analysis
- Inventory status reporting

## SQL Concepts Used
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- Aggregate Functions

## Outcome
Improved inventory data organization and reporting efficiency using structured database tables and SQL queries.
